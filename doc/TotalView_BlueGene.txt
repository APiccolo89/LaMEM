
1) Environment:

	module load UNITE totalview

2) Compiler command & flags:
 
	mpixlc_r -g -O0 -qfullpath -qkeepparm -qcheck=all

	Example:

		mpixlc_r -g -O0 -qfullpath -qkeepparm mpihello.c -o hello

3) Starting TotalView

	lltv -n <nodes> : -default_parallel_attach_subset=<rank-range> runjob -a --exe <program> -p <num>

	<program>    - executable
	<nodes>      - number of nodes 
	<num>        - processes per node
	<rank-range> - processes numbers that should start TotalView
	
		rank               - single rank 
		rank1-ranks2       - range between rank1 and rank2 (inclusive)
		rank1-rank2:stride - every rank with offset "stride" within the ranke rank1-rank2

	
	Examples:

		- one full node, start totalView on every procesor:
	
			lltv -n 1 : -default_parallel_attach_subset=0-15 runjob -a --exe hello -p 16

		- two full nodes, start totalView on every second procesor:
	
			lltv -n 2 : -default_parallel_attach_subset=0-31:2 runjob -a --exe hello -p 16
			
4) Add Program Arguments in Startup Parameters window and click OK

5) Click Go on proces window and then Yes on next window.
